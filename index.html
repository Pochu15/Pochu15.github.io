About me

My name is Carlos Pozuelo, and I am a software engineer in process. I've always wanted to create my own videogame, and, after toying around with Unity Engine for almost a year now, I've decided I should start a bigger project. Thus, Pink Shard Games was born!

Even if i'm not an expert yet, I'd like to use this blog to showcase my progress. Right now, I'm so excited to be working on Evertower Lite, my first videogame which, I hope, will come out soon.

I love learning new stuff, so, if you've got any feedback, feel free to contact me via the links below !
